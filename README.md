
# discretemath_project
Проект з дискретної математики на тему "Знаходження максимального планарного підграфу"

Консольна програма для роботи з графами та побудови планарного підграфа. Користувач задає граф у вигляді ребер, після чого програма дозволяє:

-зберігати та змінювати граф (додавання/видалення вершин і ребер);

-обхід графа в глибину (DFS) та в ширину (BFS) з побудовою дерева обходу як планарного підграфа;

-перевіряти планарність графа за необхідними умовами;

-виводити вихідний граф і отриманий планарний підграф.
